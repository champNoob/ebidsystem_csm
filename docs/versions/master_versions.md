## `v0.2` 撮合引擎

### `0.2.1` 撮合引擎初步（v1.1~v1.2）

- 引擎核心功能实现，支持完全成交

- `main.go` 装配引擎组件

- 服务层调用 `matching.Submit` 接口处理订单

不包含部分成交、撤单等

### `0.2.2` 撮合引擎（v1.3~v1.4）

- 支持部分成交

- 支持撤单

- 订单状态机

- 撮合事件 DB 事务化

- 引擎重启恢复机制
